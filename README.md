# azure-cs-json-translation
Translating a json file using Azure Cognitive Services

# Installation

```
npm install
```

# Execution

Where AZURE_CS_API_KEY is the Azure Cognitive Services API Key:

```
node index.js AZURE_CS_API_KEY ./fixtures/en.json he,fr
```
